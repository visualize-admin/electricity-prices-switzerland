## Dependencies

The application is built using the following Open Source packages

#### Programming language

- [TypeScript](https://www.typescriptlang.org/)

#### Application framework

- [React](https://reactjs.org/): Declarative component system
- [Next.js](https://nextjs.org/): React-based framework that handles bundling, routing, and server-side rendering

#### Frontend components and data visualization

- [Theme UI](https://theme-ui.com/): Design and styling system for React components
- [D3.js](https://d3js.org/): General-purpose data visualization library
- [deck.gl](https://deck.gl/): WebGL-based mapping

#### Data

- [GraphQL](https://graphql.org/): Application-centric API layer
- [Zazuko's rdf-cube-view-query](https://github.com/zazuko/rdf-cube-view-query/): Query builder for Data Cube SPARQL queries

#### Geo data

- [Mapshaper](https://github.com/mbloch/mapshaper): Conversion of Shapefiles to TopoJSON format
- [TopoJSON Client](https://github.com/topojson/topojson-client): Usage of TopoJSON files in the client

#### i18n

- [LinguiJS](https://lingui.js.org/)

#### Documentation

- [Catalog](https://www.catalog.style/)
